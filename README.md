# Penyu Airline Management System (SQL)
An end-to-end relational database system designed for an airline company, featuring 16+ interconnected tables.

# Project Overview
This project involved designing a robust database to manage airline operations, including passenger bookings, flight scheduling, staff assignments, and financial transactions.

# Database Features
- Complex Relationships: Managed 16+ tables including `Passenger`, `Flight`, `Staff`, `Booking`, and `Payment`.
- Advanced Querying: Developed 15 complex SQL queries to extract business insights, such as:
  - Identifying the most inactive months for travelers.
  - Calculating total revenue across different baggage classes.
  - Tracking frequent flyer redemption patterns.
- Data Integrity: Implemented Primary Keys, Foreign Keys, and specific data types to ensure reliable data management.

# Tech Stack
- Language: SQL (MSSQL)
- Tool: SQL Server Management Studio (SSMS)
